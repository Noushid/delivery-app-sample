## E-commerce Sample Project

Steps to run

- clone this repo.
- execute the following commands on terminal.
    - composer install
    - npm install
    - npm run dev
- create database and config in .env.

Run Following

- php artisan migrate --seed
- php artisan key:generate
- php artisan ser
